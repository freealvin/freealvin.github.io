Embed Code With Include_code

Import files on your filesystem into any blog post as embedded code snippets with syntax highlighting and a download link. In the _config.yml you can set your code_dir but the default is source/downloads/code. Simply put a file anywhere under that directory and use the following tag to embed it in a post. Syntax

File /home/Alvin/octopress/source/downloads/code/test.js could not be found
